# Curated GIF Library

Drop kid-safe GIFs here for the chat feature. Admin manages which GIFs are available.

## How to Add GIFs

1. Drop .gif files into category subfolders
2. Update the `gifLibrary` state in the ChatPage component in `index.html`
3. Each GIF needs: id, tag (category), label, and url (path to file)

## Categories

Create subfolders for each category:
- celebrate/ -- Rocket launches, confetti, party, high fives
- market/ -- To the moon, diamond hands, bull/bear
- reactions/ -- Mind blown, thumbs up, LOL, thinking
- animals/ -- Cat, dog, fish, owl animations

## Rules

- All GIFs must be kid-appropriate (ages 12+)
- No violence, profanity, or inappropriate content
- Keep file sizes under 2MB each
- Prefer looping GIFs under 5 seconds
